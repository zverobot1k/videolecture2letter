# 🎥 YouTube → Конспект (MVP1)

## 📌 Что это вообще такое

Это мой студенческий проект (но с прицелом на стартап 🚀).

Суть простая: кидаешь ссылку на YouTube-видео - получаешь готовый
конспект.

Типа вместо того, чтобы смотреть 40 минут ролик, можно за пару секунд
прочитать главное.

------------------------------------------------------------------------

## 💡 Зачем это нужно

-   нет времени смотреть длинные видео
-   хочется быстро понять суть
-   удобно делать конспекты для учебы

------------------------------------------------------------------------

## 🚀 Что уже умеет (MVP1)

-   принимает ссылку на YouTube
-   достаёт текст (субтитры или через speech-to-text)
-   обрабатывает текст
-   генерит конспект через LLM
-   работает асинхронно через очередь

------------------------------------------------------------------------

## 🏗️ Архитектура (очень просто)

\[ React \] ↓ \[ FastAPI \] ↓ \[ Redis + RQ \] ↓ [Worker](#worker) ↓ \[
Ollama / PyTorch \]

------------------------------------------------------------------------

## ⚙️ Стек

**Frontend** - React - JavaScript

**Backend** - FastAPI - Python

**AI** - PyTorch - Ollama - Whisper

**Очереди** - Redis - RQ

------------------------------------------------------------------------

## 🔄 Как это работает

1.  Пользователь вставляет ссылку
2.  Бэкенд создаёт задачу
3.  Воркер:
    -   достаёт текст из видео
    -   режет на куски
    -   отправляет в модель
4.  модель делает summary
5.  результат возвращается

------------------------------------------------------------------------

## 📡 API

### POST /summarize

{ "url": "https://youtube.com/..." }

Ответ:

{ "status": "processing", "task_id": "..." }

------------------------------------------------------------------------

### GET /result/{task_id}

{ "status": "done", "summary": "..." }

------------------------------------------------------------------------

## ⚠️ Ограничения (потому что MVP)

-   только YouTube
-   нет кэширования
-   может работать не супер быстро
-   нет авторизации

------------------------------------------------------------------------

## 🛠️ Как запустить

### Redis

redis-server

### Backend

cd backend pip install -r requirements.txt uvicorn main:app --reload

### Worker

rq worker

### Frontend

cd frontend npm install npm start
